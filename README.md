# Trinetra AI — Smart Helmet

Real-time computer vision system integrated into a wearable helmet.

## Features
- Facial recognition pipeline (OpenCV)
- AR heads-up display
- Night vision module  
- Voice command interface
- Kotlin Android companion app — zero-latency BLE control

## Stack
Python · OpenCV · YOLOv8 · Kotlin · BLE · Android SDK
